# MyHabits
Учебный проект на курсе "IOS разработчик с нуля" от Netology в блоке "Интерфейс приложения"
Что было реализовано:
- Адаптивная верстка экранов под разные устройства
- Вверстка в коде с использованием aytolayout
- Использование и настройка визуальных компонентов
- Использование UITableView, UICollectionView и кастомных ячеек
